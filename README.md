## 前言

本系列文章旨在从零开始学习游戏辅助的工作流程和开发方法，了解游戏背后的攻防对抗手段，重点侧重数据和call的追踪查找。

侧重方向在于学习整个分析过程以及思路方法，最终实现的效果和游戏不是关心的重点。

涉及到的技术有：

- windows API Hook
- dll注入
- 汇编语言
- Windows编程
- C/C++基础

没有上述的前置知识建议劝退
